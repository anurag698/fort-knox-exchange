# **App Name**: Fort Knox Exchange

## Core Features:

- User Authentication: Secure user sign-up, login, and profile management with role-based access control.
- Balance Management: Track user balances for different assets, ensuring atomicity and consistency via backend updates only.
- Order Placement: Allow users to create and manage orders, subject to balance and market validation, with options for public or private order books.
- Market Data Display: Display real-time market data including order books, trades, and price charts.
- Deposit/Withdrawal Requests: Enable users to request deposits and withdrawals, with backend-only approval and status updates.
- Admin Interface: Provide an admin interface to manage users, markets, and other exchange settings.
- Automated Content Moderation: AI tool to help flag suspicious or non-compliant withdrawal requests to aid in KYC/AML compliance

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust, security, and professionalism, aligning with financial services.
- Background color: Light gray (#F5F5F5), providing a clean and neutral backdrop.
- Accent color: Vibrant teal (#008080) for interactive elements and key calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its clean, modern, and neutral appearance.
- Use clean, minimalist icons to represent assets, order types, and other exchange features.
- Prioritize clear data presentation with well-organized tables, charts, and forms. Emphasize usability across desktop and mobile devices.
- Subtle transitions and animations to enhance user experience and provide visual feedback.
